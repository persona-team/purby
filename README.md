## Overview

Purby는 일상 일정, 날씨, 알림, 음성 상호작용을 더 따뜻하고 친근한 방식으로 연결하는 AI 스마트홈 디바이스 프로젝트입니다.

## Core Direction

- 스마트홈 디스플레이 기반의 개인화된 생활 보조 경험
- 일정, 날씨, 리마인더를 한 화면에서 확인하는 홈 대시보드
- 귀여운 캐릭터를 활용한 친근한 AI 인터랙션
- 하드웨어, 앱, 백엔드가 연결되는 통합 서비스 구조
